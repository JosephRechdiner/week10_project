The final purpose of the project is to make clients able to save their contacts in mysql database through their local host using fastapi.

The setup is very simple:
    - Got to your terminal 
    - make a directory for the program
    - Clone the github URL, "git clone https://github.com/JosephRechdiner/week10_project.git"
    - cd into it
    - Type "docker compose up" (add -d if you want the program to run in the backround)

    Thats it!
    The program is runnig!
    Tll you have to do next is, go to your browser, type: "127.0.0.1:8000" (add "/docs" if you want to use fastapi swagger)
